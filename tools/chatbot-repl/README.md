# chatbot-repl

Just a starting point...

### config
copy .env-example to .env and add your openai api key:

.env
```
OPENAI_API_KEY=<YOUR_OPENAI_API_KEY>
GPT_MODEL=text-davinci-003
```

```
npm install
```

node chatbot-repl.js
